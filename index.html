Projects

Smart Shoe for the Blind

Problem Statement: Visually impaired individuals face significant challenges while walking independently because they cannot easily detect nearby obstacles in their path.

Solution: Developed an assistive wearable device using Arduino UNO and ultrasonic sensors that continuously detects obstacles in front of the user. When an obstacle is detected within a specific distance, the system alerts the user through a buzzer or vibration motor, helping them navigate safely.

Technologies Used: Arduino UNO, Ultrasonic Sensors, Embedded C, Hardware Prototyping

ResiScan – AI Powered Resistor Scanner

Problem Statement: Identifying resistor values using color bands manually can be confusing, especially for beginners and during electronics prototyping.

Solution: Developed a web-based AI system that automatically identifies resistor color bands using computer vision and predicts the resistance value in real time. A custom dataset of more than 800 resistor images was created and used to train a MobileNetV2 deep learning model.

Technologies Used: Python, TensorFlow, MobileNetV2, Computer Vision, Image Classification

Oral Cancer Detection Using Histopathological Images

Problem Statement: Early detection of oral cancer from microscopic tissue images is difficult and requires expert medical analysis.

Solution: Developed a deep learning model using MobileNetV2 to classify histopathological images and identify potential oral cancer cases. The model was trained using a Kaggle dataset and evaluated using metrics such as confusion matrix and ROC curve.

Technologies Used: Python, TensorFlow, Deep Learning, MobileNetV2, Image Classification
